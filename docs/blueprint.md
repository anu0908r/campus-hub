# **App Name**: Campus Hub

## Core Features:

- Course Access: Provide seamless access to course listings.
- Note Repository: Allow users to upload, access, and manage study notes.
- Resource Hub: Centralize access to various study materials and external resources.
- AI Study Tool Generator: Use generative AI to suggest study tools and methods, based on a user's selected course, notes and material. Use LLM reasoning as a tool to evaluate how relevant it is for students to incorporate a new piece of data when responding to a request. 

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey a sense of calm and focus.
- Background color: Light gray (#F5F5F5), offering a clean and distraction-free environment.
- Accent color: Muted orange (#F0AB87) to highlight key interactive elements.
- Font pairing: 'PT Sans' (sans-serif) for body text to ensure readability, paired with 'Playfair' (serif) for headings.
- Use minimalist icons to represent different resources and tools.
- Implement a card-based layout for courses and resources to improve organization.
- Use subtle transitions and loading animations to enhance user experience.
